OpenClash 添加自建节点，添加订阅地址，openclash 配置文件设置，添加机场订阅教程

OpenClash 是运行在 OpenWrt 路由器上的 Clash 图形化插件，可以让全家设备科学分流、自动规则代理。

一、导入订阅

方式 1：添加订阅（推荐）
打开 服务 → OpenClash → 进入 配置订阅 → 点击「添加」 → 填入：名称：随便写 → 类型：Clash → 地址：你的订阅链接 → 保存 → 点击「更新订阅」 → 下载完成后启用

方式 2：手动上传 YAML
打开「配置管理」 → 上传 .yaml 文件 → 设置为当前配置

二、启动 OpenClash

进入「运行状态」 → 启用 OpenClash → 设置运行模式（推荐 Fake-IP） → 保存并应用

三、分流模式选择

在「运行模式」里可以选择：
🌍 全局代理（全部走代理）
🎯 规则模式（推荐）
🚫 直连模式
👉 推荐用 规则模式，自动分流国内外流量。
