VirtualBox 安装Ubuntu 25.10，下载安装教学，virtualbox安装linux系统教程

一、准备工作

首先下载 VirtualBox 和 Ubuntu ISO
1、VirtualBox下载：点击下载>>
2、下载Linux系统：点击下载>>
安装好 VirtualBox，再安装 VirtualBox Extension Pack，支持USB 2.0 / 3.0，更好的设备支持。

二、创建 Ubuntu 虚拟机



设置虚拟机的名称，添加Ubuntu ISO 文件，设置虚拟机硬件参数

开始安装 Ubuntu 系统

安装完系统后，点“立即重启”

切换软件镜像源，国内阿里云速度比较快

安装增强功能，可以开启3D加速，使用也更方便
安装完增强功能后，关闭虚拟机，再打开虚拟机设置，在显示里开启3D加速，将显存改为128MB

✅ 总结

VirtualBox 安装 Ubuntu 只需要：
1️⃣ 下载 VirtualBox
2️⃣ 下载 Ubuntu ISO
3️⃣ 创建虚拟机
4️⃣ 加载 ISO
5️⃣ 安装系统
6️⃣ 安装 Guest Additions 增强功能
